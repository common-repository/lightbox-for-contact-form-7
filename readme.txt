Shows Contact Form 7 in a fancy lightbox.

== Description ==

Lightbox for Contact Form 7 helps you to show [Contact Form 7](https://wordpress.org/plugins/contact-form-7/) in a fency lightbox. .

You will need to build a shortcode using [cf7lightbox form_id='' title=''] where 'form_id' and 'title' will be contact form 7 form 'id' and 'title'.
